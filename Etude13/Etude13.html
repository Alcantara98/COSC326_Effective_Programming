Yes there is always a distinct largest number for each base that has no repeated digits.
The largest possible digit in a given base of 'b' is b - 1. The further left a digit is, the higher its value,
hence given all possible digits in the given base which are between (inclusive) 0 and b-1,
we place the largest digit to the left, then consecutively place the next largest digit after,
to its right until all possible digits have been used. So Given base 7,
the largest would be 654321, or 6543210 if we consider 0 as a digit.
